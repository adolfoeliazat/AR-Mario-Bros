AR. Mario Bros.

This project overlays Mario Bros on an image taken by your webcam

Usage

To run this project on your computer, you must download this and add it to the "libraries" in your development environment. Additionally, you must have a built-in webcam in order for the project to work.

Libraries Used

  • Sarxos Webcam
  • Canny Edge Detection

Made by @arnav-gudibande and @d2shan for CS I Final Project
